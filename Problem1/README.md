## Problem 1

Given a `p` and a string `str`, return true if `s` follows the pattern, else false.

<br>

**Example 1:**

<pre>
<b>Input:</b> p = "abba", str = "dog cat cat dog"
<b>Output:</b> true
</pre>

**Example 2:**

<pre>
<b>Input:</b> p = "abba", str = "dog cat cat fish"
<b>Output:</b> false
</pre>

**Example 3:**

<pre>
<b>Input:</b> p = "aaaa", str = "dog cat cat dog"
<b>Output:</b> false
</pre>
